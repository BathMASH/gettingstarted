Getting started with LaTeX

This is one of four videos and associated resources developed using funding provided by the sigma Network. It has been reviewed for submission to MathCentre/StatsTutor.

Video

This video and associated documents enable getting started with LaTeX using an online, collaborative LaTeX system. LaTeX is free and can be installed on Windows, Mac or Linux computers. If you wish to install LaTeX on your own computer you can find further information on doing so at:
